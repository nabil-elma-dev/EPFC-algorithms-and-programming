# EPFC-algorithms-and-programming
This course introduces the Java programming language, helping students become familiar with conditional statements and loops. It also includes chapters on list and string manipulation. Finally, it introduces the fundamental concepts of object-oriented programming, including classes, objects, methods, and constructors.
